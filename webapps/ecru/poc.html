Course Reserves Unleashed!
Proof-of-Concept

Course Reserves Unleashed! consists of the ecru (electronic course reserves unleashed) database, an instance of Apache Solr 4.*, which backs a robust RESTful API pattern set for querying the books, articles, and weblinks that are "reserved" for a particular course.

At Harvard, a job runs nightly that updates the ecru database with data from the Reserves List System (aka the "Reserves Tool"), but the underlying software is designed to support reserves data from other sources as well.

This page is a "proof-of-concept": using AJAX to query the database via the APIs, and processing the JSON response with JavaScript (specifically, the jQuery library).
